# متجر الحدث موبايل

هذا مجلد جاهز للرفع يحتوي على:
- index.html (واجهة احترافية جاهزة)
- images/ (صور افتراضية مؤقتة للأجهزة يجب استبدالها بالصور الواقعية لاحقًا)

## الخطوات
1) استبدل الصور داخل مجلد images بصورك الواقعية مع الحفاظ على نفس أسماء الملفات.
2) أنشئ وسيط تيليجرام (Google Apps Script Web App) لحماية التوكن.
3) افتح index.html وابحث عن السطر:
   const TELEGRAM_PROXY_URL = "";
   ثم ضع رابط النشر للويب آب.
